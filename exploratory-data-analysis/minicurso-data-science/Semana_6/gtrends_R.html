Hoje vamos ver como funciona o pacote GtrendsR.

O Google Trends uma ferramenta do google que mostra o volume de pesquisa de determinadas palavras em relao ao total de pesquisa. O resultado um ndice de 0 a 100 e quanto mais prximo de 100, maior o interesse pela palavra.

Para exemplificar o funcionamento, vamos verificar se a busca pela palavra seguro-desemprego aumentou durante a pandemia.

Para comear, vamos instalar e carregar o pacote gtrendsR. Vamos carregar o pacote tidyverse que iremos usar posteriormente para anlise.

Cabe notar que existem 3 elementos essenciais para a funo gtrends: i) palavra ou frase de interesse (keyword) ii) regio de interesse (geo) iii) perodo de tempo de interesse

Vamos procurar primeiro pelas buscas por seguro-desempregro no Brasil. Vejamos atravs do exemplo abaixo. Note que, de acordo com a explicao acima, precisamos definir a palavra buscada (seguro-desemprego), a regio de interesse, no caso, Brasil (geo=c(BR)) e o perodo desejado (nesse caso, queremos todo o perodo disponvel - time = all).

segurodesemprego <- gtrends(c("seguro-desemprego"), geo = c("BR"), time = "all")

Podemos plotar diretamente os resultados usando a funo plot:

plot(segurodesemprego)

Vamos fazer uma busca por algumas palavras durante a pandemia no google trends. Para isso, vamos criar uma funo (adaptada do Michael, do Casual inference), que vai facilitar a elaborao dos nossos grficos. Essa funo ser plot_trend, com o argumento keyword_string. Vejamos abaixo.

#funo adaptada de Michael - Casual Inference
#funo para plotar os grficos a partir de gtrends, para o perodo de janeiro de 2020 a junho de 2020.
#ns vamos inserir uma linha vertical prximo a data de incio da "quarentena", usando geom_vline no dia 10 de maro de 2020. 
plot_trend<-function(keyword_string){
data<-gtrends(c(keyword_string), time= "2020-01-01 2020-06-14", geo = "BR")
time_trend<-data$interest_over_time %>%
  mutate(hits=ifelse(hits=="<1",0.5,hits),
         date=as.Date(date),
         keyword=factor(keyword, levels = keyword_string))
plot<-ggplot(data=time_trend, aes(x=date, y=as.numeric(hits), colour=keyword)) +
  geom_line() +
  geom_vline(xintercept = as.numeric(as.Date("2020-03-10"))) +
  theme_bw() +
  scale_y_continuous(breaks = NULL) +
  theme(legend.position = "bottom",
        legend.title = element_blank(),
        legend.text=element_text(size=10),
        axis.title.x=element_blank(),
        axis.title.y=element_blank())
return(plot)
}

A funo acima ir nos ajudar a fazer os grficos mais rapidamente. Vamos procurar os termos como cortar o cabelo, como fazer po, como fazer caf e treino em casa, a fim de verificar se houve alguma mudana nas buscas a partir da pandemia.

cabelo_plot<-plot_trend(keyword_string = c( "como cortar o cabelo"))
comida_plot<-plot_trend(keyword_string = c("como fazer po"))
fitness_plot<-plot_trend(keyword_string = c("treino em casa"))
coffee_plot<-plot_trend(keyword_string = c("como fazer caf"))

(cabelo_plot + comida_plot) / (fitness_plot + coffee_plot)
